Installation fuer Windows:
1) Die .NET-Runtime kann bei https://dotnet.microsoft.com/en-us/download/dotnet/5.0[Microsoft] heruntergeladen werden.
2) GTK+ 3.24 kann beispielsweise https://github.com/tschoonj/GTK-for-Windows-Runtime-Environment-Installer/releases/download/2022-01-04/gtk3-runtime-3.24.31-2022-01-04-ts-win64.exe[hier] heruntergeladen werden.
3) Speichern der Batch-Datei als #InstallUpdateCsbp.cmd in einem leeren Verzeichnis, z.B. D:\Haushalt\CSBP.
4) Starten von cmd.exe.
5) Wechseln in das Verzeichnis:
 D:
 cd \Haushalt\CSBP
6) Ausfuehren der Datei #InstallUpdateCsbp.cmd.
7) Beim naechsten Mal starten mit #Csbp.cmd.
