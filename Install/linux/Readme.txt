Installation fuer Linux (getestet fuer Ubuntu 20.04):
1) Das Paket dotnet-runtime-5.0 installieren.
2) GTK+ 3 ist schon installiert.
1) Speichern der Batch-Datei als #InstallUpdateCsbp.sh in einem leeren Verzeichnis, z.B. /opt/Haushalt/CSBP.
2) Starten einer Terminal-Sitzung.
3) Wechseln in das Verzeichnis und Ausfuehrungsberechtigung setzen:
 cd /opt/Haushalt/CSBP
 chmod 755 #InstallUpdateCsbp.sh
4) Ausfuehren der Datei: ./#InstallUpdateCsbp.sh.
5) Beim naechsten Mal starten mit ./#Csbp.sh.
